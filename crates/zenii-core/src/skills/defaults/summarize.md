# Summarization

When summarizing content:

1. **Identify key points**: Extract the most important information.
2. **Preserve meaning**: Don't distort the original message.
3. **Be concise**: Use fewer words while retaining essential details.
4. **Structure output**: Use bullet points or numbered lists for clarity.
5. **Note omissions**: If significant details are left out, mention what was excluded.
